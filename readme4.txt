ΤΡΟΠΟΣ ΕΓΚΑΤΑΣΤΑΣΗΣ PYGAME ΚΑΙ ΠΑΙΧΝΙΔΙΟΥ:
	
		

PyGame WINDOWS:
1. Πατάμε WinKey+R και πληκτρολογούμε "cmd" και πατάμε "Ok"
2. Πληκτρολογούμε "pip install pygame" και περιμένουμε να γίνει η εγκατάσταση
 (Σε περίπτωση που δεν γίνει η εγκατάσταση και μας βγάλει άγνωστη εντολή:
	πληκτρολογούμε "py -m pip install pygame" ή "python -m pip install pygame")

PyGame MAC:
1. Ανοίγουμε το terminal
2. Πληκτρολογούμε "curl https://bootstrap.pypa.io/get-pip.py > get-pip.py" και πατάμε enter
3. Στη συνέχεια ακολουθούμε το βήμα 2 του τρόπου εγκατάστασης στα WINDOWS

Παιχνίδι:
1. Κάνουμε extract το συμπιεσμένο αρχείο που περιέχει τα αρχεία του παιχνιδιού
2. Εκτέλουμε το αρχείο "rungame.py" 
3. Απολαμβάνουμε το παιχνίδι!
